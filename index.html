Question no 5: Demonstrate the concept of inline CSS and external CSS.

Inline CSS

Inline css is the method of adding styles in HTML elements using the "style" attribute. This attribute allows to apply the CSS directly to the HTML element. Here's an example:

<p style="color: red; font-size: 16px;"> This is inline CSS </p>

In the above example, the "style" attribute is used to apply two CSS rules to the <p> element: color:red and fontsize:16px.

External CSS

External CSS is a method of adding styles to HTML documents by linking to a separate CSS files. The CSS file contains all the styles for the page and is linked to the HTML document using <link> element. Here's the example:

<head> <link rel="stylesheet" href="styles.css" </link> </head>

In the above example, "link" element tells the browser where to find the external CSS file. The "href" attribute specifies the path to the css file, and the "rel" attribute tells the browser that this is a stylesheet. Now, when you open this HTML document in the web browser, it will apply the styles defined in the external CSS file to the elements in the document.

Question no 5: Demonstrate the concept of Class selector and ID selector in CSS.

Class Selector

Class selector is used to select elements with a specific class attribute. The class attribute can be used on any HTML element. Here's an example:

<h1 class="test--class"> This is a heading </h1> <p class="test--class"> This is a paragraph </p>

In the above example, the "class" attribute is used to add a class to the <h1> and <p> elements. Now, we can use the class selector to style these elements. Here's the example:

.test--class { color: red; font-size: 24px; }

In the above example, the "." symbol is used to select elements with a specific class attribute. The "test--class" class is used to style the <h1> and <p> elements respectively.

ID Selector

ID selector is used to select elements with a specific unique id attribute. The id attribute can be used on any HTML element. Here's an example:

<h1 id="heading"> This is a heading </h1> <p id="paragraph"> This is a paragraph </p>

In the above example, the "id" attribute is used to add an id to the <h1> and <p> elements. Now, we can use the id selector to style these elements. Here's the example:

#heading { color: red; font-size: 24px; } #paragraph { color: blue; font-size: 16px; }

In the above example, the "#" symbol is used to select elements with a specific id attribute. The "heading" and "paragraph" ids are used to style the <h1> and <p> elements respectively.
