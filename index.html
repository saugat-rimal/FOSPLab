Why not frame-based layout?

Frames are an older web design technique and may not be compatible with all modern browsers.The frame tag was used in HTML 4 to define one particular window (frame) within a frameset. It may be worth considering alternative layouts, such as using CSS to create a responsive layout that adjusts to different screen sizes.
